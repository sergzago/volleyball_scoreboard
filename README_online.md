# Онлайн результаты матчей

Новая страница `online.html` предоставляет возможность просмотра живых результатов матчей, которые проходят сегодня, но еще не завершены и не попали в архив результатов.

## Возможности

- **Реальное время**: Изменения счета отображаются синхронно с изменениями на странице управления
- **Фильтрация**: Фильтрация по типу игры (классический/пляжный волейбол) и турниру
- **Автообновление**: Возможность настроить автоматическое обновление данных
- **Группировка**: Игры автоматически группируются по типу и турниру
- **Адаптивный дизайн**: Страница корректно отображается на устройствах с разным размером экрана

## Формат отображения

- **Команды**: Названия команд в первом столбце
- **Общий счёт**: Второй столбец показывает итоговый счёт матча
- **Текущий сет**: Третий столбец показывает счёт текущего сета
- **Сеты**: Последующие столбцы показывают счёт по завершённым сетам

## Типы игр

- **Классический волейбол**: До 3 побед в сетах, до 25 очков в сете (15 в 5-м)
- **Пляжный волейбол**: До 2 побед в сетах, до 21 очка в первых двух сетах (15 в третьем)

## Использование

1. Откройте страницу `online.html` в браузере
2. Выберите нужный тип игры и турнир в фильтрах
3. Настройте интервал автообновления при необходимости
4. Наблюдайте за изменениями счета в реальном времени

## Технические особенности

- Страница использует Firebase Realtime Database для получения данных
- Автоматически определяет игры, которые проходят сегодня
- Исключает из отображения уже завершённые матчи
- Поддерживает до 5 сетов для классического волейбола и до 3 сетов для пляжного
